# BitScout

Minimal Bitcoin wallet tracker built with Expo/React Native and the mempool.space API.

## Run locally

1. Set `EXPO_PUBLIC_BASE_URL` in `.env`.
2. Install dependencies with `yarn install`.
3. Run `npx expo prebuild`.
4. Launch the iOS app with `yarn ios`.
